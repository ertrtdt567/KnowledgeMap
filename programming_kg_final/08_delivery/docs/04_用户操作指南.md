# 编程领域知识图谱模块用户操作指南

## 1. 使用对象

- 教师或项目验收人员：查看五门课程的知识组织、来源和题目映射。
- 学习者：检索知识点、进入课程子图、查看关联概念和练习题。
- 维护人员：在不改变正式版本的前提下核对数据来源、版本与 Neo4j 查询结果。

## 2. 打开系统

按《部署运维手册》启动 API 与前端后，在浏览器打开 Vite 提供的本地地址。页面标题为“编程知识图谱”，顶部显示当前位置，左侧为搜索和统计，中央为图谱画布，右侧为节点详情。

## 3. 课程图谱浏览

1. 首页展示“编程领域”下的五门课程。
2. 单击课程或知识节点，右侧显示其名称、类型、置信度、来源数量、别名和关联节点。
3. 单击“进入子图”，查看该节点下属知识领域、知识单元和知识点。
4. 使用顶部“返回上级”或面包屑回到课程图或根图。
5. 使用放大、缩小、适应视图按钮观察密集子图；可拖动节点辅助查看关系。

## 4. 知识点检索

1. 在左侧“搜索知识点”输入名称、别名或关键字。
2. 在结果中选择知识点，系统跳转到其所属图谱并自动选中该节点。
3. 搜索结果若为题目，选择后直接打开题目窗口。

建议优先使用课程名、知识点名称或常用缩写检索，例如 `Java`、`多态`、`UML`、`链表`。

## 5. 查看节点详情与关联

- **别名**：用于展示缩写、同义表达或译名；不代表下属知识点。
- **关联节点**：单击可跳转至一跳关联概念或资源。
- **来源**：表示支撑该节点的课程材料数量；具体资料和页码以节点数据属性为准。
- **代码示例**：选择代码示例节点后可加载原题或原始代码内容。

## 6. 使用习题

1. 在知识点详情中，若存在“X 道练习”按钮，点击后打开习题窗口。
2. 阅读题干、代码和选项；选择“查看答案”查看已核验答案。
3. 使用“上一题”“下一题”浏览同一知识点关联题目。
4. 搜索到题目时可直接打开该题，不影响当前课程图位置。

题目映射表示该题直接考查的知识点，不表示题目只能使用这一项知识。当前正式题库包含 45 道题、73 条直接考查映射。

## 7. 验收演示建议顺序

1. 从根图展示五门课程。
2. 任选一门课程进入层级子图，说明“课程局部知识树 + 全局核心概念映射”的结构。
3. 搜索一个跨课程概念，例如“类”或“接口”，展示节点详情和关联课程知识。
4. 打开一个代码示例，展示原题来源。
5. 打开一个知识点练习，展示题干、答案与知识点映射。
6. 在 Neo4j Browser 执行统计查询，说明 JSON、Neo4j 和 RDF 的同版本交付。

## 8. 使用限制

- 当前为本机演示版，应在 API 和前端均启动后使用正式数据。
- 大型整图受浏览器布局影响，建议通过课程图和子图进行展示，而非一次性渲染全部节点。
- 节点和题目结果以 `v2026.07.18` 正式发布目录为准；候选数据不在前端正式展示范围内。

## 5. 已知限制与后续建议

1. 当前为 Windows 本机演示部署，未声明为公网生产服务。
2. 课程知识点复核中 3 条层级问题已记录；为避免破坏已定型前端，本发布版本不再调整，作为下一版本修订项。
3. 前端正式验收前应核对 API 实际读取的是 `v2026.07.18`，而不是模拟数据或历史图谱。
4. 后续新增课程或题库时，应重新走预处理、抽取、规范化、课程树、质量核验和版本发布流程，不直接编辑正式 JSON。
